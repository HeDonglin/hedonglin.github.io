发表于 2017-06-18 | 分类于 前端基础 | | 阅读次数

在不同浏览器中嵌套问题：

块元素

  1. 默认独占一行
  2. 没有宽度时，默认撑满一行
  3. 支持所有css命令
  4. 几个特殊的块元素只能嵌套内联元素，不能再嵌套块级元素：h1~h6,p,dt,caption
  5. 宽高内外边距可控
  6. 可容纳内联及块元素

内联元素

  1. 同排可以继续跟同类的标签
  2. 内容撑开宽度
  3. 不支持宽高
  4. 不支持上下的margin
  5. 代码换行被解析，会被解析成空格

内联块元素

  1. 可在同一行显示
  2. 宽高可控
